# Note Redactor (Qt Widgets)

MVP редактора нотного стана на C++/Qt с ООП-структурой.

## Что уже реализовано

- 5 линий нотного стана.
- Добавление нот разных длительностей (whole/half/quarter/eighth) по клику.
- Добавление разделителя тактов (тонкая вертикальная палка).
- Постановка на линию и между линиями (привязка к шагам стана).
- Перетаскивание выбранной ноты только влево/вправо.
- Перетаскивание выбранной вертикальной палки только влево/вправо.
- Перемещение выбранной ноты вверх/вниз на 1 шаг (кнопки и стрелки).
- Автоматическая отрисовка дополнительных мини-линий для нот выше/ниже стана.
- Удаление ноты (кнопка, клавиша Delete или инструмент Eraser).
- Класс `Note` хранит музыкальные свойства: обозначение, октаву, частоту.

## Сборка и запуск

```bash
cmake -S . -B build
cmake --build build -j
./build/NoteRedactor
```

## Управление

- В верхней панели:
  - `Select` - выбрать символ
  - `Whole note` - поставить ноту (целая)
  - `Half note` - поставить ноту (половинная)
  - `Quarter note` - поставить ноту (четвертная)
  - `Eighth note` - поставить ноту (восьмая)
  - `Treble clef` - поставить скрипичный ключ (SVG)
  - `Rest` - поставить паузу (SVG)
  - `Sharp` - поставить диез (SVG)
  - `Flat` - поставить бемоль (SVG)
  - `Bar line` - поставить вертикальную палку (разделитель тактов)
  - `Eraser` - удалить символ кликом
- Выбранную ноту можно:
  - двигать мышью по горизонтали
  - двигать по высоте кнопками `Move up` / `Move down`
  - двигать клавишами `Up` / `Down`
  - удалить `Delete`
  - `Treble clef`, `Rest`, `Sharp`, `Flat` можно двигать мышью по X и по высоте (привязка к шагам)

## Как добавить свои SVG-символы

1. Положи SVG-файлы в папку `assets/svg/` (например: `treble_clef.svg`, `rest_quarter.svg`, `sharp.svg`, `flat.svg`).
2. Добавь файл в `resources/symbols.qrc` с alias в формате `assets/svg/<name>.svg`.
3. В `src/view/StaffWidget.cpp` в функции `svgResourceForType(...)` пропиши путь ресурса (например `:/assets/svg/your_symbol.svg`).
4. Если нужен новый тип символа:
   - добавь тип в `MusicSymbol::SymbolType` и `ToolType`,
   - добавь кнопку в `SymbolPaletteWidget`,
   - добавь создание символа в `mousePressEvent`,
   - добавь отрисовку и размер `targetRect` в `drawSymbols`.
